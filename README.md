### 📖 About the Project
This repository contains a collection of practical exercises developed during my **Technical Course at COTEC**. It focuses on fundamental programming logic and data manipulation using **Python**.

### 📂 Repository Structure
The project consists of several scripts solving core programming problems, including:
*   **Mathematics & Conversion:** Final grade calculations, unit conversions (meters), and currency exchange (USD).
*   **Geometry:** Area and perimeter calculations.
*   **Financial Logic:** Salary calculations, pricing, and inventory management logic.
*   **Utilities:** Temperature converters and automated multiplication tables.

### 🛠️ Tech Stack & Skills
*   **Language:** Python 3.x
*   **Concepts:** Algorithms, variables, control structures, and basic automation.
