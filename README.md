## 💻 Atividade de Linguagem de Programação - COTEC

**Repositório dedicado ao armazenamento de exercícios práticos desenvolvidos durante o curso técnico da COTEC, com foco em lógica de programação e manipulação de dados utilizando Python**

**🚀 Tecnologias Utilizadas**

* ***Linguagem:*** Python 3.x

* ***Ambiente de Desenvolvimento:*** Lógica de programação e algoritmos

**📂 Estrutura do Repositório**

O projeto consiste em uma série de scripts que resolvem problemas fundamentais, incluindo:

* **Matemática & Conversão:** Cálculos de média final, conversão de metros para outras unidades e câmbio de dólar.

* **Geometria:** Cálculos de área e perímetros.

* **Lógica Financeira:** Cálculo de salários, preços e valores de peças.

* **Utilitários:** Conversores de temperatura e tabuadas automáticas.
